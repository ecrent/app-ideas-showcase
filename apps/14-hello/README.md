# Hello

> Tier 1 app from [florinpop17/app-ideas](https://github.com/florinpop17/app-ideas)

## Description

A simple, delightful greeting app that displays "Hello" in 15 different languages. Browse through languages using the Previous/Next buttons or click any language tile to jump directly to it. Each language includes a country flag emoji for instant visual context.

Perfect for learning basic greetings across cultures or simply enjoying the visual variety of different writing systems.

## Features

- 15 languages: English, Spanish, French, German, Italian, Portuguese, Dutch, Swedish, Polish, Russian, Japanese, Chinese, Korean, Arabic, and Hebrew
- Smooth navigation with Previous/Next buttons
- Quick-select grid of language tiles
- Responsive design that works on mobile, tablet, and desktop
- Animated flag emoji for visual appeal
- Country flag indicators for cultural context

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite

## How to Use

1. **Navigate with buttons** — Click **Previous** and **Next** to browse through languages
2. **Quick select** — Click any language tile in the grid to jump directly to that greeting
3. **Keyboard shortcuts** — Use left/right arrow keys for quick navigation
4. **Visual cues** — Country flag emojis provide instant cultural context
5. **Responsive design** — Works seamlessly on mobile, tablet, and desktop
