# Dynamic CSS Variables

> Tier 1 app from [florinpop17/app-ideas](https://github.com/florinpop17/app-ideas)

## Description

A live CSS custom variables editor for developers. Create, edit, and preview CSS variables in real-time with a split-view interface showing the editor on the left and live preview on the right.

## Features

- **Live Editor** — Add, edit, and remove CSS variables instantly
- **Color Picker** — Built-in color picker for color values
- **Live Preview** — See changes applied to UI components immediately
- **CSS Import** — Paste CSS code to automatically extract and import variables
- **Copy Output** — Generate and copy clean CSS output for use in your projects
- **Responsive Design** — Works seamlessly on desktop and tablet screens

## How to Use

1. Add new variables with the "+ Add Variable" button
2. Edit the variable name and value in the input fields
3. Use the color picker for color values
4. Paste CSS code in the "Import from CSS" section to extract variables
5. Copy the generated CSS output to use in your project
6. Preview how your variables look on the right panel

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
cd apps/10-dynamic-css-variables
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
