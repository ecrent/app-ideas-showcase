# Color Cycle

> Tier 1 app from [florinpop17/app-ideas](https://github.com/florinpop17/app-ideas)

## Description

An interactive color cycling animation application. Watch the background smoothly transition through vibrant colors while you control the speed and direction. Features play/pause controls, quick-select color buttons, and a customizable animation speed slider.

## Features

- 🎨 Smooth color cycling with 8 vibrant colors
- ⏯️ Play/pause controls for the animation
- ⚡ Adjustable animation speed (0.5s to 3s)
- ◀️▶️ Manual previous/next color navigation
- 🎯 Quick color selection buttons
- 📊 Real-time color display (hex and name)
- 🎭 Animated card with subtle pulsing effect during playback

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
cd apps/06-color-cycle
npm install
npm run dev
```
