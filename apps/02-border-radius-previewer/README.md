# Border Radius Previewer

> Tier 1 app from [florinpop17/app-ideas](https://github.com/florinpop17/app-ideas)

## Description

An interactive CSS border-radius previewer that lets you adjust border-radius values in real-time and see the changes instantly. Perfect for experimenting with rounded corners on your web designs.

### Features

- **Live Preview** — See border-radius changes in real-time as you adjust values
- **Unit Toggle** — Switch between pixels (px) and percentages (%)
- **Sync All Corners** — Quickly set all corners to the same value
- **Individual Corner Control** — Fine-tune each corner separately (top-left, top-right, bottom-right, bottom-left)
- **Preset Values** — Quick shortcuts for common shapes (Sharp, Rounded, Pill, Circle)
- **CSS Output** — View and copy the generated CSS code with one click
- **Reset** — Easily reset to default values

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
cd apps/02-border-radius-previewer
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## How to Use

1. Use the **Unit** toggle to choose between `px` and `%`
2. Adjust values using:
   - **Sync All Corners** input to set all corners at once
   - **Individual corner inputs** for precise control
   - **Preset buttons** for common shapes
3. Watch the preview update in real-time
4. Copy the generated CSS code to use in your project
