# Flip Image

> Tier 1 app from [florinpop17/app-ideas](https://github.com/florinpop17/app-ideas)

## Description

A simple and intuitive image flip tool that allows you to horizontally or vertically flip any image. Upload an image, apply flip transformations, and download the result instantly. Perfect for quick image manipulations and simple photo edits.

## Features

- 📤 Easy image upload via drag-and-drop style interface
- 🔄 Horizontal and vertical flip transformations
- 👁️ Real-time preview of original and flipped images
- 💾 One-click image download
- 🎨 Clean, responsive UI built with Tailwind CSS
- ⌨️ Support for all common image formats (PNG, JPG, GIF, etc.)

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- HTML5 Canvas API

## Getting Started

```bash
cd apps/12-flip-image
npm install
npm run dev
```

## How to Use

1. Click the upload area or select a file to upload your image
2. Click "Horizontal" or "Vertical" buttons to flip the image
3. Combine both flips for a 180-degree rotation
4. Click "Download Image" to save the flipped result
5. Click "Clear" to start with a new image
