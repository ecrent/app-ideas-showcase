# Dollars to Cents

> Tier 1 app from [florinpop17/app-ideas](https://github.com/florinpop17/app-ideas)

## Description

A simple and elegant dollar-to-cents converter. Enter a dollar amount and instantly see the equivalent value in cents. Perfect for quick conversions, learning currency mathematics, or understanding currency relationships.

## Features

- **Real-time conversion**: Convert dollar amounts to cents instantly
- **Input validation**: Clear error messages for invalid inputs
- **User-friendly interface**: Clean, modern design with Tailwind CSS
- **Keyboard support**: Press Enter to convert
- **Clear function**: Quick button to reset and start over
- **Helpful guide**: Built-in explanation of how the converter works

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
cd apps/09-dollars-to-cents
npm install
npm run dev
```

## How to Use

1. Enter a dollar amount in the input field (e.g., 5.50)
2. Click the "Convert" button or press Enter
3. View the result in cents (e.g., 550¢)
4. Use "Clear" to reset and start a new conversion
